# Full-fledged-login-system

Complete authentication app that lets user sign up, sign in, reset password, sign in using social accounts.

## Features
* Sign Up
* Sign In
* Sign In using social accounts
* Reset passwords

## Tech I used:

* Bootstrap
* NodeJS
* Express
* EJS
* Passport-local
* Passport-Google-Oauth
* Mongoose
